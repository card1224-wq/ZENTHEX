Zenthex 고객센터

계정, 구독, Studio 생성, Trading 실거래 키 인증 문제를 확인하는 공간입니다. 정식 오픈 전에는 대표 검토용 안내 페이지로 사용합니다.

계정/로그인

  • 로그인은 이메일 주소로 진행합니다.
  • 비밀번호 찾기는 힌트 질문과 이메일 인증 코드로 재설정합니다.
  • 배포 후 계정이 사라진 것처럼 보이면 영구 DB 연결 상태를 확인해야 합니다.

구독/결제

  • Studio Pro, Trading Pro, Ultimate 플랜으로 나뉩니다.
  • 정식 결제는 월 자동결제 구조로 연결합니다.
  • 영수증과 결제내역은 마이페이지에서 확인합니다.

Studio

  • 비구독 사용자는 하루 1회 보기 전용 체험입니다.
  • 다운로드는 Studio Pro 또는 Ultimate에서 열립니다.
  • 생성 실패 시 프롬프트, 토큰, AI Provider 상태를 확인합니다.

Trading / Upbit

  • Secret Key가 점으로 보이는 것은 보안 입력칸이라 정상입니다.
  • 보기 버튼으로 복사 상태를 확인한 뒤 키 인증을 진행할 수 있습니다.
  • 업비트 키에는 자산조회와 주문하기 권한, 서버 공인 IP 허용이 필요합니다.
