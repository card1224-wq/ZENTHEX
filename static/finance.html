업비트 전체 KRW 코인에서 먼저 24시간 강한 후보를 거르고, 실제 진입은 1분·3분·5분 단타 신호로 판단합니다. 거래량 급증, 단기 고점 돌파, 1분 이동평균 추세, 최근 모멘텀을 점수화하고 과열·급락 후보는 제외합니다.

전략 설정

실거래 모드는 로그인 후 Trading Pro 또는 Ultimate 구독 상태에서만 표시됩니다.

실거래 모드는 실제 돈으로 매수/매도합니다. 업비트 API 키는 조회/주문 권한만 허용하고 출금 권한은 절대 켜지 마세요.

키가 정상인데 실패하면
업비트 Open API 키에 이 서버의 공인 IP가 허용되어 있어야 합니다. 권한은 자산조회와 주문하기가 필요하고, 출금 권한은 켜지 마세요. Secret Key는 발급 직후 한 번만 보이므로 틀렸다면 새 키 발급이 가장 안전합니다.

바이낸스 실거래는 다음 연결 단계입니다. 현재 화면에서는 방향과 전략 체험만 제공하고, 실제 주문은 연결 완료 후 로그인·구독·API 키·위험동의 뒤에 열립니다.

10%, 30%, 50%는 단타 목표가 아니라 장기·고위험 목표입니다. 엔진이 오래 보유할 수 있고 손실 위험도 커집니다.

실거래에서는 업비트 계정의 사용 가능한 KRW 현금을 기준으로 투자합니다. 체험 모드는 가상 잔고 기준입니다.
