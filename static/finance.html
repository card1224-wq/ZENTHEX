투자 위험 고지

Zenthex Trading은 자동매매 도구이며 투자 자문 또는 수익 보장 서비스가 아닙니다. 실거래는 실제 업비트 주문이며, 출금 권한이 없는 API 키만 사용하세요.

연결 상태 / 공식

Upbit 체험/실거래 연결 Binance 다음 실거래 연결 대상

전체 KRW 코인을 스캔하고 1분·3분·5분 모멘텀과 거래량 급증을 점수화합니다. 직접 선택 모드는 전체 스캔 없이 빠르게 진입합니다.

Upbit 허용 IP

Upbit Open API 허용 IP에는 사용자 휴대폰/PC IP가 아니라 Zenthex 실거래 서버의 공인 IP를 등록해야 합니다.

Secret Key가 점으로 보이는 것은 정상입니다. 복사 상태를 확인할 때만 보기 버튼을 누르세요.

키가 정상인데 실패하면
업비트 Open API 키에 이 서버의 공인 IP가 허용되어 있어야 합니다. 권한은 자산조회와 주문하기가 필요하고, 출금 권한은 켜지 마세요. Secret Key는 발급 직후 한 번만 보이므로 틀렸다면 새 키 발급이 가장 안전합니다.

바이낸스 실거래는 다음 연결 단계입니다. 현재 화면에서는 방향과 전략 체험만 제공하고, 실제 주문은 연결 완료 후 로그인·구독·API 키·위험동의 뒤에 열립니다.

KRW 현금 전액은 업비트 원화 잔고만 사용합니다.

비율은 KRW 현금 잔고 중 일부만 쓰는 방식입니다.

고급 전략 설정 추적익절 · 보유코인 정리 · 위험 설명

고정 목표는 선택한 수익률에 도달하면 바로 매도 후 종료합니다. 10%, 30%, 50%는 단타가 아니라 장기·고위험 목표입니다.

이미 코인을 사서 KRW가 거의 없으면 KRW 전액/비율 모드는 주문 가능 금액 부족으로 진행되지 않습니다.

보유 코인 정리 후 전액 재진입은 업비트에 이미 들고 있는 KRW 마켓 코인을 먼저 시장가 매도해 KRW로 만든 뒤, 새 후보 코인에 다시 들어가는 고위험 모드입니다.
