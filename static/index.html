스튜디오가 무엇인가요?

Zenthex Studio는 말과 도면을 3D 결과물로 바꾸는 SaaS 도구입니다.

  • 프롬프트를 최적화해 3D 생성 요청을 만듭니다.
  • 2D 도면은 벽과 윤곽을 분석해 GLB로 변환합니다.
  • 무거운 작업은 서버 워커에서 처리하고 화면에는 진행률과 결과를 보여줍니다.
