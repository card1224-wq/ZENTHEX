Zenthex란?

Zenthex는 AI 3D 제작과 자동매매 실행 구조를 하나의 계정, 구독, 모바일 제어 흐름으로 제공하는 SaaS 플랫폼입니다.

  • Studio 체험은 보기 전용이며 다운로드는 구독 후 제공합니다.
  • Trading 체험은 구조 확인용이며 API 키 입력을 요구하지 않습니다.
  • 실거래는 로그인, 구독 권한, 위험 동의, API 키 확인 후에만 실행됩니다.
